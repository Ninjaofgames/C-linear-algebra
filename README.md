## Features
- Identity matrix & null matrix
- Random matrix (using Mersenne Twister or mt19937)
- Compare two matrixes
- Adding two matrixes
- Scalar multiplication
- Check triangular matrixes
- Transpose
- Trace
- Minor matrixes
- Determinant
- Multiplying two matrixes (knowed in Python as matmul())
